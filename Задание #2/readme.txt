Task 1

Калькулятор с использованием callback: 
Напишите функцию calculate, которая принимает два числа 
и callback-функцию для выполнения операции над этими числами. 
Функция calculate должна передать эти числа в callback-функцию и 
вернуть результат.

Task 2

Генератор HTML-ссылок: Напишите функцию generateLink, 
которая принимает URL и текст ссылки, а также callback-функцию. 
Функция generateLink должна создать HTML-ссылку, используя предоставленные 
URL и текст ссылки, а затем передать эту ссылку в callback-функцию
